####
CREATE A SPRING BOOT PROJECT
####
CREATE THE PROJECT WITH THE FOLLOWING STRUCTURE:
src/main/java/com/example/myapp
├── MyappApplication.java (Main class)
├── config/              (Configuration classes, e.g., security, database)
├── controller/          (Handles HTTP requests and responses)
├── dto/                 (Data Transfer Objects for API requests/responses)
├── entity/              (JPA Entities for database mapping)
├── exception/           (Custom exceptions and global handlers)
├── repository/          (Interacts with the database)
├── service/             (Contains business logic, defines interfaces)
└── util/                (Helper/utility classes)
THE PROJECT WILL BE ABOUT A CODING WEBSITE WHERE USERS CAN REGISTER, LOGIN, AND VIEW CODING TUTORIALS.
THE PROJECT WILL INCLUDE THESE FEATURES:
"- Đăng nhập, Đăng Ký, Quên mật khẩu 
- Viết code , kiểm tra code , Chọn ngôn ngữ
- Tích xu thông qua giải các bài code, Đổi thưởng, Điểm danh hằng ngày
- Bình luận , chia sẻ hướng giải quyết bài toán
- Tìm kiếm bài code, Thông báo , 
- Cập nhật thông tin người dùng
- Chatbox gợi ý hướng giải quyết bài toán
- Chấm điểm dựa trên thời gian chạy, bảng xếp hạng 
- Cuộc thi hàng tuần"

#### TECHNOLOGIES TO BE USED

the project will use the following technologies:
- Spring Boot
- Spring Security
- Spring Data JPA
- Judge0 for running and evaluating code submissions
- Hibernate
- MySQL or PostgreSQL
- Maven or Gradle for build management
